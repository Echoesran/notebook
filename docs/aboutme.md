
# 人生守则清单

- 1.极度坦诚。
- 2.管理欲望：尤其不要纵容掌控欲、拖延欲，尽量避免与人争论。
- 3.对全世界祛魅。
- 4.做对的事，把事做对。
- 5.不要让重要的事变成紧急的事。
- 6.知行合一。
- 7.从微小的行动开始。
- 8.不要偏听偏信，自己验证。
- 9.和谁一起工作、生活，这很重要。
- 10.知道自己的具体的问题，从解决那一个问题开始。不要幻想问题，不要奢求别人写的攻略。
- 11.尽量审慎地说话，不说不知道的话。
- 12.正确的选择不会纠结。
- 13.follow your heart。
- 14.做不做一件事的判断标准很简单，不做会不会后悔、做了有什么代价，简单估算ROI。
- 15.一些问题可以先不解决，交给时间和命运，前提是要持续做事和思考。
- 16.表达感谢和喜欢是人类最佳文明。
- 17.建立框架思维，包括人生到具体业务场景。
- 18.永远不要陷入短视频的邪恶陷进，少刷小红书和微博，少玩手机求你了。保持用脑卫生。
- 19.得到是最好的祛魅方式，不要在没有真正得到的时候轻易根据认知做出判断，再根据认知判断选择。
- 20.与人为善，广结善缘。
- 21.珍惜时间价值。
- “根据目标估算出时间的价值，就是做选择的机会成本，每一分钟都是算得出来的。无法做决策主要是因为没有锚定物，不知如何计算。如果你按照时间投向与机会成本来构建思维系统，那么大事小事无非就是大资源小资源的计算而已。拖延、执行力差，本质是没想清楚、算清楚。”
- 22.既怕何必想，既想何必怕。
- 23.多看事实，少看观点。solid。
- 24.一切都是可以争取的，从让世界看到你的面孔，到让世界听到你的声音，再到让世界感受到你的存在和价值。
- 25.热爱命运，全情投入，玩好这场游戏。
- 26.想解决问题，就要站在别人的角度，从别人的利益出发来帮自己完成目标。
- 27.不要骄傲，不要沾沾自喜自己身上的标签。
- 28.social聊天的时候做输出型的人，即学会聊别人的话题。
- 29.不要害怕犯错，对错误归根究底。
- 30.80%的精力应该放在20%的事情上。
